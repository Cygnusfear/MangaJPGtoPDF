Manga JPGtoPDF
===========================



Converts Manga JPGs PDF. Spreads are automatically rotated to landscape orientation. Sorts by 000 numbering.

**Usage:**

depends on FPDF:
`pip install FPDF`

run: 
`python main.py {image directory}`

**Extra:**

- Rename cover file to 'cover.jpg'
